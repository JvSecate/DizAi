# Sobre o projeto
DizAi é um sistema de feedback que utiliza IA, em especifico análise de sentimento em textos, para automaticamente catalogar e gerar uma avaliação baseado em um feedback. Este é um projeto didático  utilizado em sala de aula.

## Informações
Este repositorio é referente ao front-end do projeto. Para ver o repositório referente ao backend visite "[DizAi-Backend](https://github.com/JvSecate/DizAi-Backend)"
